# dbc 

tinyodbc의 포팅이다. 파일들을 나눠서 유지보수하기 수월하도록 변경하고 
result set 메모리 관리 효율을 높여 성능을 올렸다. 

아직 테스트해야 할 내용은 많이 있다. db 처리는 매우 안정적이어야 하고 
예외처리도 깔끔해야 한다. 되도록 서버가 죽지 않도록 하는 것이 중요하다.
